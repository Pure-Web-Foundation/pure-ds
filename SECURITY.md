# Security Policy

## Supported Versions

This project is in early development. We aim to support the latest minor release on the main branch.

## Reporting a Vulnerability

Please report security issues privately.

- Open a private security advisory in GitHub (Security > Advisories) if available, or
- Email the maintainers (see the repository owner profile)

We will acknowledge receipt within 72 hours and work with you to assess and remediate the issue. Once resolved, we will disclose responsibly with credit if desired.
