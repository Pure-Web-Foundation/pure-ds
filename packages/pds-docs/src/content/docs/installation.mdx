---
title: Installation
description: Install @pure-ds/core in your project.
---

```sh
npm install @pure-ds/core
```

After install, PDS automatically sets up your project:

- Creates `pds.config.js` with a starter configuration
- Configures the MCP server for VS Code and Cursor
- Copies AI guidance files (`.github/copilot-instructions.md`, `.cursorrules`)

Then run the build to generate static assets:

```sh
npm run pds:build
```

See [Getting Started](/getting-started) for the full setup walkthrough.
